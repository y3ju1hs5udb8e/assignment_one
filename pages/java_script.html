Introduction


JavaScript (Js) is the scripting language for the web based application invented in 1995 by Brendan Eich. It is a cross-platform, Object Oriented Programming (OOP) language used to make webpages interactive by adding functionality such as theme toggle script, popup menus, and many others. Js isn't just applicable in just front-end but aswell in backend, Node.js, which allows users to add advance functionality like connecting front-end (client side) with database via Application Programming Interface (API).
Js contains a standard libary of objects, such as Array, Map, and Math, and like other programming languages it uses elements such operators, control structures, and statements. Core Js can be extended for a varity of purposes such as:
–Client-side: Js uses Document Object Model (DOM) to manipulate the HTML elements to intreact based on the user. For example, client-side extensions allow an application to place elements on an HTML form and respond to user events such as mouse clicks, form input, and page navigation.
–Server-side: Initally Js was developed to work on front-end but with the introduction of Node (Node.js), it was able to run out of browser making it suitable for server side rendering. For example, Server-side extensions allows an application to communicate with a database.

Fun-Fact: In 1997 A.D Js became an European Computer Manufacturers Association (ECMA) standard and began to known as ECMAScript. ECMAScript's versions are denoted by ES and ES6 is the common and most used Js version.

Basic Syntax


Js is a scripting language used to add interactivity and dynamic behavior to web pages. Well it can be applied into HTML files directly using script tag or externally making .js file. Well both needs to be in a script tag but there is a difference on that, let's see the differenceses:

1. Internal Js

As the name speaks, internal method is directly written inside the .html file using the script tag. The script tag can be placed inside the head of the HTMl like embedded CSS or just before the closing body tag (</body>)
In head method: 

                        
                            <head>
                                <title>Internal JS Example</title>
                                <script>
                                    function greet() {
                                        alert("Hello from Internal JavaScript!");
                                    }
                                </script>
                            </head>
                            <body>
                                <button onclick="greet()">Click Me</button>
                            </body>

Before closing body tag: 
                        
                            <body>
                                <button onclick="greet()">Click Me</button>

                                <script>
                                    function greet() {
                                        alert("Hello from Internal JavaScript!");
                                    }
                                </script>
                            </body>


2. External Js

The external approach required the seperated script file with .js file type. The using the script tag it is linked in HTML.
index.html 

                        
                            <head>
                                <title>Internal JS Example</title>
                                <script src="script.js"></script>
                            </head>
                            <body>
                                <button onclick="greet()">Click Me</button>
                            </body>

script.js 
                        
                            function greet() {
                                alert("Hello from Internal JavaScript!");
                            }


Well, when clicking both examples button it eventually will alter user saying, "Hello from Internal JavaScript!". It's because of the script. As you can see above the script kinds looks C-like language. So let's break it down:
  • function: It is a kewword used to declare function in js.
  • greet(): the greet is the name of the functon while () is used to add parameters which a function might use
  • {}: The curly braces enclose the body of the function. Everything inside them will run when the function is called.
  • alert(): alert is the built-in Js funtion which display message in browsers alert box. In this case, it will alert saying, "Hello from Internal JavaScript!"

Conclusion


JavaScript is a crucial programming language that gives websites dynamic behavior and interactivity. From basic alert boxes to sophisticated theme toggles and animations, it enables developers to create captivating user experiences, whether it is stored in a separate.js file (external JavaScript) or embedded directly within HTML using the <script> tag (internal JavaScript). For small scripts or demos, internal JS is quick and practical, but external JS provides better code organization, scalability, and reusability. Writing clear, maintainable, and interactive web applications starts with an understanding of both approaches.
