Introduction

In there modern era, using plain and traditional .css file gets heavy for larger projects. The term heavy here stands for lines of codes written to decorate a page, making it responsive, and many more Cascading Style Sheets (CSS) related tasks. Traditional CSS get's annoying overtime, that's why frameworks were introduced. A framework consists of reusable software components that provide a foundation for developing applications. Similary in CSS, to reduce the annoying situation there are lots of CSS frameworks such as, Bootstrap, Tailwind CSS, Bulma, and many more.
Bootstrap, it is a free and open-source front-end framewrok for making web applications. It is also considered CSS-toolkit for developing responsive and mobile-first websites by providing a collection of syntax for template designs. The Bootstrap is built on Htper Text Markup Language (HTML), CSS and Java Scripts (Js) which makes it include the basics for responsive web development, so developers only need to insert the code into a pre-defined grid system. Also in computing, the term bootstrap means to boot or to load a program into a computer using a much smaller initial program to load in the desired program, which is usually an OS.

Why use bootstrap?


Bootstrap is a flexible and dynamic front frame that provides a free collection of tools for creating websites and web applications. Developers or designers should use bootstrap in their projects because:

  • It is a framework which will help developers & designers design websites quickly and easily.
  • It includes HTML and CSS based templates for forms, buttons, tables, image carousels, etc. It also provides the support for JavaScript plugins
  • It is adaptable and simple to work with in all range of projects.
  • It responds with design, maintains wide browser compatibility, provides consistent design through usable components, and is very easy to use and quick to learn.

Tutorial


Its is quite simple to implement bootstrap in our projects. You can even follow the offical document of Bootstrap

But let's make a simple one here. Steps:

1. Creating a file (.html file)

If you're using Vs-code, then simply create a .html file and type !, to use the basic sinppit. In step one that much is enough.

2. Linking Bootstrap

We basically have to link, the css and the js into our project. It's like injecting steroid into out body. The simple thing is, we have to now link it into our HTML file using the CDN links, copy form the offical docs site. (Link: CDN links). Now
in file representation (.html file) 

                        
                            <-- in head section add .css -->
                            <head>
                                <meta charset="utf-8">
                                <meta name="viewport" content="width=device-width, initial-scale=1">
                                <title>My project</title>
                                <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.5/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-SgOJa3DmI69IUzQ2PVdRZhwQ+dy64/BUtbMJw1MZ8t5HZApcHrRKUc4W0kG879m7" crossorigin="anonymous">
                            </head>
                            <body>
                                <-- in body (brfore closing) section add .js -->
                                <h1>Hello, world!</h1>
                                <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.5/dist/js/bootstrap.bundle.min.js" integrity="sha384-k6d4wzSIapyDyv1kpU366/PK5hCdSbCRGRCMv+eplOQJWyd1fbcAu9OCUj5zNLiq" crossorigin="anonymous"></script>
                            </body>
                            </html>
Add those links, as shown in the code above.

3. Start using Bootstrap

Head to offical Bootstrap site and navigate to search bar, and search required designe template. Copy it and paste it in you .html file. For example: 

                        
                            <body>
                                <-- paste bootstrap code here -->
                                <ul class="nav nav-tabs">
                                    <li class="nav-item">
                                    <a class="nav-link active" aria-current="page" href="#">Active</a>
                                    </li>
                                    <li class="nav-item">
                                    <a class="nav-link" href="#">Link</a>
                                    </li>
                                    <li class="nav-item">
                                    <a class="nav-link" href="#">Link</a>
                                    </li>
                                    <li class="nav-item">
                                    <a class="nav-link disabled" aria-disabled="true">Disabled</a>
                                    </li>
                                </ul>
                            </body>
As shown in the above code, copy the bootstrap code (in my case it's of tabbar) and paste the bootstrap inside body tag and enjoy.

Conclusion


A common and powerful front-end framework, Bootstrap makes it easier to create responsive and mobile-first websites. Its flexible a grid system, already developed CSS classes, and JavaScript components allows developers to quickly create layouts that look professional without having to start from scratch. Bootstrap greatly cuts down on development time while maintaining design consistency, whether you're creating a small personal project or a large-scale application. Learning Bootstrap guarantees that your website looks fantastic on all devices and speeds up your workflow.
